# Dipesh Bist Portfolio

Personal portfolio website for Dipesh Bist, a Full Stack Developer based in Kathmandu, Nepal.

## Stack

- Vite
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- EmailJS contact form

## Local Development

```sh
npm install
npm run dev
```

The development server runs on `http://localhost:4000`.

## Build

```sh
npm run build
```

## Content

The site highlights production IoT platform work, AI/RAG projects, full-stack systems, Shopify customization, and Android development experience from Dipesh's resume.
